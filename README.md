> phpCloud是一个云函数api开发框架，主轻量级、面向对象、路由映射。

## 主要特性

* 支持 [腾讯云]
* 轻量级
* 面向对象
* 控制器、Request、Response实现
* Composer支持

## 安装

~~~
composer create-project kyour-cn/phpcloud phpcloud 0.*
~~~
